> 本文内容：Windows 电脑常用软件的常用快捷键记录

# Windows

检索：Ctrl + F

打开 emoji：Win + .  或者Win + ;

打开搜索：Win + Q

显示桌面：Win + D

打开资源管理器：Win +E

打开设置：Win + I

锁定屏幕：Win + L

打开任务视图：Win +Tab

视图切换：ALT+TAB

显示粘贴板：Win + V

资源管理器：Ctrl + Shift + ESC

安装 11 系统时使用 mailto:no@thankyou.com 作为用户名，可以跳过联网账户，直接创建本地账户 mailto:no@thankyou.com

# R

运行选中的代码：Ctrl + Enter

注释/取消注释：Ctrl + Shift + C

撤销：Ctrl + Z

取消撤销：Ctrl + Shift + Z

快速添加 section 标签：Ctrl +Shift + R

获取当前路径 `dirname(rstudioapi::getSourceEditorContext()$path)`

到达某路径 `setwd(path)`

# MATLAB

注释：Ctrl + R

取消注释：Ctrl + T

打开某代码:：Ctrl+D

停止运行代码：Ctrl+C

测试运行代码：F9

debug 模式下步进：F10

debug 模式下步进，但是会进入到子函数内再步进：F11

在 MATLAB command 命令打开 aaa.m 文件，`edit aaa.m` 或者 `open aaa.m` 都可以

`cd ..\\\\..`   back to the two last folder

`cd ..`   back to the last folder

查找编码格式 `slCharacterEncoding()`

获取当前路径 `pwd`

到达某路径 `cd path`

# Visual Studio Code

注释/取消注释：Ctrl + /

打开当前文件夹内的其他文件：Ctrl+P

切换选项卡：Ctrl+ Shift+Tab

切换侧边栏<kbd> Ctrl+B

打开设置：Ctrl+,

打开终端：Ctrl+J

打开命令面板：Ctrl+ Shift+P

运行代码：F5

复制当前行：Shift + Alt +up/down

删除当前行：Shift + Ctrl + K

行增加缩进：Ctrl + [

行减少缩进：Ctrl + ]

# Sublime Text

注释/取消注释：Ctrl + /

# Notepad++

注释：Ctrl + Q

取消注释：Ctrl + Shift + Q

# Word

输入空方框：先输入数字 2610 ，再按 Alt+X 键

输入带 √ 的方框：先输入数字 2611 ，再按 Alt+X 键

输入带 × 的方框：先输入数字 2612 ，再按 Alt+X 键

仅粘贴文本：Ctrl + Shrift + V

# Excel

单元格内换行：ALT + Enter

# Obsidian

工作区：Ctrl + W

编辑模式切换：Ctrl + E

打开设置：Ctrl + ，

打开命令面板：Ctrl +P

Admonitions：Ctrl +P，然后输入 `ad`

多窗口打开：Ctrl + 点击笔记

纯文本格式粘贴：Ctrl + Shift +V

# Notion

新窗口打开该页面（需要选择一个页面）：Ctrl + Shift +Enter

# Zotero

打开折叠的条目或者文件夹：Shift + =

# 浏览器

新标签页打开网页链接：Ctrl +单击
